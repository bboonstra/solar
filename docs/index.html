SOLAR

Semi-Autonomous Outdoor Life Assisting Robot

About

A research-driven robotics project exploring sustainable, autonomous plant care through sensor data collection and environmental interaction.

Key Features

  • Modular sensor and actuator integration
  • Real-time environmental data collection
  • Autonomous decision-making
  • Custom-fabricated components
  • Raspberry Pi powered
